## 🌍 Overview

**SHAPEthiopia** is a full-stack humanitarian platform designed to:

- 📚 Improve **education access**
- 💧 Deliver **clean water solutions**
- 👩 Empower **women & youth**
- 🏘 Strengthen **local communities**

> 💡 Built with modern web technologies + AI to maximize impact.
